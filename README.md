# Facebook-Clone
Live: https://facebookclone111.netlify.app

# Installing

1. Clone the latest code on the master branch
2. Install dependencies

`npm install`

3. Run the web server!

`npm start`

# Usage

Navigate to http://localhost:3000 on your browser
